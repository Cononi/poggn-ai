# Quick start

```bash
python3 .codex/script/setup_links.py
```

Check language.

```text
$codex-language status
```

Start MAW.

```text
$maw spring boot shop order payment rest api swagger jpa
```

Use shortcuts for state to avoid model-token waste.

```text
$codex-state summary --for-ai
$codex-task trace --for-ai
```

## SAW minimal-token example

```text
$codex-saw suggest --text "fix dto field mapping"
```
